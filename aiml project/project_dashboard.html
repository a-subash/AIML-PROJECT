Promo Code Abuse Detection using AIML
Promo codes are widely used by digital platforms to attract users, but they’re often misused. This project uses Machine Learning (Isolation Forest) to detect suspicious users based on IP reuse, gibberish names, and promo overuse. The system improves fraud prevention, campaign fairness, and platform trust.
